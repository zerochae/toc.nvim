toc.nvim HTML Demo

Open this file and run :Toc to see headings, links, images, blockquotes, code, and list items in the panel.

Overview

The HTML parser reads structure from treesitter when available.

Usage

Opening the panel

Tip: the panel mirrors your document outline.
require("toc").setup {}
:Toc

Keymaps

  • Enter jumps to the entry
  • q closes the panel
  • See the project page

Elements

Headings

All of h1 through h6 are indexed.
